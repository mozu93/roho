# リリースノート

## v1.3.0 — 2026-07-23

### 還付金振込

- 「還付金振込」タブを追加しました。年度ごとに還付金額を入力し、登録済みの振込先口座を確認できます。
- 選択した事業所について、宛名ラベルと還付金振込通知書PDFを出力できるようにしました。
- 振込指定日を都度入力して、全銀協規定の総合振込データ（120バイト固定長）を出力できるようにしました。
- 設定タブに、還付金の振込元口座を登録する項目を追加しました。

### 手数料計算

- 一覧の表示列・列固定・並び替えを名簿／年度更新タブと揃えました。
- 概算保険料の連続入力を追加し、画面に表示されている順に入力できるようにしました。
- 督促メール用のチェック選択、ラベル出力、対応履歴検索を追加しました。

### メール送信

- Microsoft 365の代理差出人アドレスを設定できるようにしました。
- トークンキャッシュの暗号化、レート制限時の再試行、添付容量チェック、連続送信エラー時の中断を追加しました。

## v1.2.26 — 2026-07-21

### 年度更新

- 枝番別提出状況に保険番号を表示し、確認日を直接編集・保存できるよう改善しました。
- 提出状況の選択肢から「対象外」を削除しました。
- 年度更新メモの入力欄を拡大し、メモ・提出状況・確認日・全体状況の変更日を記録するよう改善しました。
- 一覧の「最終更新日（年度更新）」列を末尾へ移動しました。

### 画面構成

- タブ順を「名簿、年度更新、手数料計算、委託解除、設定」に変更しました。
